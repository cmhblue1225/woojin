스마트폰 교체 관련 주의사항
A기기로 출결하다가 학교 전자출결에 로그인 한 적 없는 새기기, B로 로그인하면 바로 출결사용이 가능하며, A기기는 사용 불가함
다시 A기기로 변경 사용하려면 A, B 모두 7일간 사용하지 말아야 하며 8일째 1개(A또는B)만 사용 가능
A기기로 출결하다가 학교 전자출결에 로그인 한 적 있는 기기, C(예를들어 본교 다른학생핸드폰을 구입해서 사용할 경우)를 사용하면 로그인이 안되며, 이 때 A는 가능함.
C를 사용하려면 A와 C 모두 7일간 사용하지 않고 8일째 되는 날 1개(A 또는 C)만 사용해야 함
자신의 핸드폰기기를 쓰다가 친구 기기를 사용하면 로그인이 안되며, 7일 단말보안정책도 적용 안함. 자신의 기기로 다시 사용하면 바로 로그인 사용 가능
1주일 막힐 경우 담당교수님에게 연락하여 전자출결 변경 요청해야 하며, 1주일 경과 후 하나만 정하여 전자출결 사용해야 함
전자출결의 학생 출결변경은 학기 중 담당 교수님만 전자출결 변경이 가능함

iOS인 경우 OS 버전 11 이상 지원
유의사항
OS 버전이 낮아서 위치 출결 불가능한 학생의 경우 교수님이 직접 확인하여 출결 변경해 주거나(강의가 출강 되었을 경우만), 인증 번호 출석으로 가능합니다. 수업 후에도 출결 수정 가능합니다.
스마트폰 블루투스 사용고지 및 위치서비스(GPS) 사용 여부
전자출결 앱에서는 블루투스를 사용하여 출결/출강을 합니다. 그러므로 스마트폰의 블루투스로 음악이나 통화를 자주 사용하시거나 모바일 핫스팟을 사용하시는 분은 제조사에 따라 오작동이 발생 할 수 있습니다. 그러한 경우 모바일 핫스팟을 사용 중지해 주시거나 스마트 폰 종료 후 재시작 하시면 정상 작동합니다. 강의실에서 출결 정상 작동 여부는 전자출결 앱 실행 후 환경설정 매뉴에서 신호상태확인을 클릭하여 확인할 수 있습니다.
구글 및 애플사의 정책으로 블루투스 사용시 반드시 위치서비스(GPS)가 ON으로 설정 되어 있어야 합니다. 단, 위치서비스(GPS)는 스마트폰 OS에서만 체크하며 전자출결 앱에서 위치 정보를 사용하거나 수집하지 않습니다.
스마트폰 보안 정책(부정출결방지)
전자출결앱으로 학번을 입력하여 최초 로그인 하신 경우 자동으로 보안 정책이 적용됩니다.
최초 전자출결 앱으로 로그인 하신 경우 7일 동안 다른 스마트 폰으로 변경하여 로그인하실 수 없습니다.
자신 학번외에 다른 학번으로 절대 로그인 하지 마십시오.
출결 진행 방법 및 출결 시간 확인
전자 출결 앱 홈 화면에서 반드시 출석하기 버튼을 클릭하여 진행하시면 블루투스가 작동하여 출결이 진행됩니다.(블루투스가 동작하여 위치 검색을 하므로 스마트폰 기종에 따라 3초~5초 정도 소요됩니다.)
전자 출결 앱 홈 화면에서 지난 강의가 표시되는 경우 리플래시 버튼을 클릭하여 화면을 갱신 바랍니다.
전자 출결 시스템의 시간을 기준으로 출결이 체크 되오니 반드시 유의바랍니다.
출석 → 수업 시작후 15분까지,
지각 → 수업 시작후 15분부터 20분까지
스마트 폰의 시간을 강제로 수동으로 설정할 경우 전자출결 앱 사용을 제한합니다.
인증번호 출석
교수님이 강의 중 인증번호 출석 진행할 경우 전자출결 앱에서 인증번호를 입력하거나 블루투스를 이용하여 출석 확인합니다.
인증 번호 출석 진행 할 경우 푸시 메시지가 전송되며, 전송이 되지 않을 경우 전자출결 메인 홈에서 리플래시 버튼을 클릭하면 확인이 가능합니다.
휴강이 될 경우 학생 출결 데이터
휴강이 되는 경우 학생 출결 데이터는 있지만 인정 되지 않습니다.
강의실 문제로 휴/보강 등록 없이 강의실이 임의 변경 되는 경우
시간표에 표시된 지정 강의실에서 출결, 출강 진행한 후 다른 강의실로 이동.
2시간 이론/2시간 실습(실외, 실내)이 이루어 지는 경우 출결
첫교시에 지정 강의실에서 출결 처리해야 합니다.
실습실에도 위치 비콘 설치 되어 있으므로 지정강의실에서 첫 교시에만 출결을 체크하시면 가능합니다.
강의 야외에서 수업하는 경우
사전에 야외 비콘을 별도로 단과대 교학팀에 비치하며 담당교수 신청 시 교수명, 교과목명 관리자 등록 후 제공하여 사용
인터넷 강좌 및 집중이수제 교과 외 일부 전자출결 시스템 제외 교과는 종이 출석부로 대체

교수님의 결강
스마트 폰, 모바일에서 출강하기 하지 않은 경우 결강으로 처리
결강이 되더라도 학생 출결 상황은 그대로 표시됩니다. 그러나 결강이 되면 학생 출결을 변경할 수 없습니다.
결강이 되면 학생의 출결을 수정할 수 없으므로 학사 지원팀의 관리자 출강 승인이 필요합니다.
